# Integrating External AI API and Prompt Engineering PDF

This plan addresses the requirement to connect the `@ai` chat command to a real AI API (Groq/OpenAI), pass the full content of the rich text editor to the AI as context, and generate a PDF documenting the prompt engineering used for this process.

## User Review Required
> [!IMPORTANT]
> **API Key Setup:** To make the AI integration work, we will need an API key. I will write the code to use the `GROQ_API_KEY` environment variable. You will need to make sure this environment variable is set when running the backend.
>
> **Python Dependency:** The integration relies on having Python installed on the server to execute the API call and generate the PDF. I will use a Python script invoked by the C backend to handle the HTTPS API requests, as making raw HTTPS API calls directly in C requires heavy libraries like OpenSSL/libcurl which breaks the zero-dependency constraint of this project.

## Proposed Changes

### 1. Python AI Helper Script
We will create a helper script that the C backend will call to perform the API request.

#### [NEW] [ai_helper.py](file:///c:/Users/307520%20B7IIN/OneDrive/Desktop/prompton/ai_helper.py)
- A Python script that accepts two arguments: the `document_content` and the `user_prompt`.
- It will construct a system prompt instructing the AI to act as an assistant for the collaborative editor, providing the document content as context.
- It will call the Groq API (or another HTTP endpoint) and print the response to stdout.

### 2. C Backend Modification
We will modify the C backend to invoke the Python script and read its output instead of returning hardcoded responses.

#### [MODIFY] [main.c](file:///c:/Users/307520%20B7IIN/OneDrive/Desktop/prompton/src/main.c)
- Locate the [handle_ws_message](file:///c:/Users/307520%20B7IIN/OneDrive/Desktop/prompton/src/main.c#647-971) function where `@ai` commands are processed.
- Instead of the `if/else` hardcoded responses, write the document content (from `d->content`) and the user query to temporary files or pass them securely to the `ai_helper.py` script via `popen()`.
- Read the AI's response from the script's stdout and broadcast it back to the chat.

### 3. Prompt Engineering PDF Generation
We will create a script to generate the requested PDF documenting the prompt strategy.

#### [NEW] [generate_prompt_pdf.py](file:///c:/Users/307520%20B7IIN/OneDrive/Desktop/prompton/generate_prompt_pdf.py)
- A Python script using a library like `fpdf` to generate a formatted PDF.
- The PDF will explain the exact system prompt used, the rationale behind it, and how the rich text content is injected into the context window for the AI.
